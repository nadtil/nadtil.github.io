Motor Controller PCB

A motor controller was developed to operate and control automatic blinds that can be commanded wirelessly through a mobile phone. For this, a state of the art control system was developed using experimental data and MATLAB. Finally this control system was implemented on an embedded system.

Given below is the PCB of the motor controller used to control the speed and direction of the motor rotation. Input can be a PWM signal which is used to control the current fed into the motor. Main supply is 12V DC and the board itself contains step down DC-DC converters to provide 5V and 3.3V for external electronic circuitry.
