Zigbee Smart Switch PCB

This is a smart switch designed to connect to the user's phone through Zigbee protocol. By using this smart switch the user can control their lights wirelessly. This integrates with the existing wall-switches from Clipsal providing the user with a seamless experience between the physical light switch and the mobile. Whenever the user switch on the light using the physical switch it will be reflected on their mobile and vice versa.
